# Blockchain Metadata Chain of Custody Evidence Verification

A TypeScript-based project for verifying evidence integrity using blockchain-style metadata and chain-of-custody tracking. The system ensures that digital evidence can be traced, validated, and protected from tampering through structured verification logic and metadata tracking.

---

## Project Overview

This repository provides a backend/system that focuses on:

* Evidence metadata storage and verification
* Chain-of-custody tracking
* Integrity validation of digital evidence
* API-based verification workflow

It is designed to simulate or implement a secure evidence tracking system using blockchain-inspired principles.

---

## Tech Stack

* TypeScript
* Node.js
* PNPM (package manager)
* API-based architecture

---

## Project Structure

```
lib/            # Core logic (verification, metadata handling)
scripts/        # Utility and setup scripts
artifacts/      # Generated or stored evidence outputs
attached_assets/ # Supporting assets
```

## Installation

### 1. Clone the repository

```bash
git clone https://github.com/MalakArab3245/Blockchain-Metadata-Chain-of-Custody-Evidence-Verification-.git
cd Blockchain-Metadata-Chain-of-Custody-Evidence-Verification-
```

### 2. Install dependencies

Using pnpm:

```bash
pnpm install
```

Or using npm:

```bash
npm install
```

---

## Running the Project

### Development mode

```bash
pnpm dev
```

### Build project

```bash
pnpm build
```

### Start production

```bash
pnpm start
```
